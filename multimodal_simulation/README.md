# Multi-modal Simulation


# Prerequisites
Please refer to "env_dynamic_multimodal.yml" 

# Usage
For SGLD, please run:
```
python3 lorenz_sgld.py
```
or 
```
source main_lorenz.sh
```


For cycSGLD, please run:
```
python3 lorenz_cycsgld.py
```


For SGLD, please run:
```
python3 lorenz_resgld.py
```
